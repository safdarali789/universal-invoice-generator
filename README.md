# Invoice Generator

A free, multi-language invoice generator supporting international formats (GST, VAT, Proforma, Receipt) with live preview and PDF export.

## 🔗 Live Demo

https://safdarali789.github.io/universal-invoice-generator/

## ✨ Features

- **6 Invoice Templates** — Standard, GST/Tax Invoice, VAT, Proforma, Freelancer, Receipt
- **3 Invoice Languages** — Urdu, English, Arabic (with full RTL support)
- **15+ Currencies** — PKR, USD, EUR, GBP, INR, AED, SAR, and more
- **Auto Calculations** — Tax, discount, and shipping computed live
- **PAID / DUE Stamp** — Visual status seal on the invoice
- **Bank Details Section** — For payment instructions
- **Print / Save as PDF** — One-click export
- **Responsive Layout** — Works on desktop, mobile, and embedded iframes (e.g. Google Sites)
- **No Backend Required** — Pure HTML/CSS/JS, works entirely in the browser
- **SEO Optimized** — Meta tags, Open Graph previews, robots.txt, and sitemap.xml included

## 🔘 On/Off Toggle Switches

| Toggle | Effect when ON | Effect when OFF |
|---|---|---|
| **Tax** (GST/VAT) | Adds a tax line to totals | Removes tax row from the invoice |
| **Discount** | Adds a discount line | Removes discount row from the invoice |

## 🚀 How to Use

1. Open `index.html` in any browser (or visit the live link above)
2. Fill in your company details, client details, and line items on the form
3. Choose the invoice language (Urdu / English / Arabic) for the document itself
4. Click **"Print / Save as PDF"** to export

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript — no frameworks, no build step, no dependencies
- Fonts: Source Serif 4, Inter, JetBrains Mono, Noto Nastaliq Urdu (via Google Fonts)

## 📦 Deployment

- **GitHub Pages** (this repo): Settings → Pages → source set to `main` branch
- **Google Sites**: Upload to Google Drive → Share publicly → Embed by URL
- **Netlify / Vercel**: Drag-and-drop deploy, no configuration needed
